# sistema_proyecto

## librerias:
- flask
- bootstrap 4
- flask-mysqldb
## Añadir imformación de la base de datos
1. crear un archivo llamado .env
2. copiar y remplazar los datos:

```
HOST_SQL='hosting_de_la_db'

USER_MYSQL='usuario'

PASSWORD_MYSQL='contraseña'

DB_MYSQL='nombre_base_de_datos'
```
